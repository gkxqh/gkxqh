树莓派安装Windows ARM
注意：

1.安装的是ARM架构的Windows！

准备工具及系统镜像

  1. Windows on Raspberry imager

下载页面：https://www.worproject.ml/downloads#windows-on-raspberry-imager

官方提供Google Drive的下载链接。

  1. Windows ARM系统镜像

方法一：访问UUP dump网站（https://uupdump.net/）或UUP (Unified Update Platform) Generation Project（https://uup.rg-adguard.net/）（以前者为例）。![]
\树莓派安装win11\1.webp

选择想要安装的系统版本，如想要尝试Windows 11（Windows on ARM 11添加了对x64应用的支持），则选择“Latest Dev Channel build”，否则选择“Latest Public Release build”即可。记得要点选对应版本右侧的“arm64”按钮。

方法二（推荐）：访问NEXT, ITELLYOU（https://next.itellyou.cn/），注册并登录，下载ARM架构的镜像（Win 10或Win 11）

选择系统的语言及勾选想要的系统版本后（如Windows Home或Windows Pro，勾选其一即可），Summary页面的选项保持默认即可。下载压缩包并解压，以管理员权限运行uup_download_windows.cmd文件。下载需要等待一段时间，同时保证电脑硬盘中有足够的空间（5GB左右）。

写入系统镜像

使用刚刚下载的Windows on Raspberry imager将系统镜像写入SD卡，依次选择SD卡的盘符（注意不要选错了）、刚刚下载打包生成的系统镜像、以及由软件自动下载驱动以及UEFI固件后，就可以写入了。

启动并初次设置Windows系统

将SD卡插入树莓派并插电启动，出现树莓的界面后，需要耐心等待一段时间，树莓图案的下方会出现旋转的等待圆环，表示系统正在启动。

之后就能正常进入系统：

初次启动Windows 11时，系统可能需要强制下载更新并安装。这段时间会比较久，也请耐心等待。

PS：解除树莓派4（Pi4/Pi400)内存默认限制为3G的解决方法：

在插电启动时，一直按“ESC”键，直到进入了UEFI的设置界面。之后导航至

“Device Manager -> Raspberry Pi Configuration -> Advanced Configuration”页面，将“Limit RAM to 3 GB”的选项设定为“Disabled（禁用）”。

在这个设置页面同时还能设置CPU的频率，建议将CPU频率调至“Max”，应该能在一定程度上改善系统运行的性能和使用体验。
